# nagz

Nagz is a family-oriented nagging/reminder iOS app.

## Documentation
- `nagz/Docs/REQUIREMENTS.md`
- `nagz/Docs/PREFERENCES.md`
- `nagz/Docs/ARCHITECTURE.md`

## Xcode
Open `nagz.xcodeproj`.

The `nagz/` group is filesystem-synchronized, so docs in `nagz/Docs/` are visible in the Xcode project navigator.
